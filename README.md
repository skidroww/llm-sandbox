# LLM Sandbox 

수업 내용 복습 및 개인적으로 LLM을 이것저것 테스트해보기 위해 만든 연습용 레포지토리입니다. 

##  포함된 내용
* **LangChain & Vector DB**: `langchain.ipynb`, `vectorDB.ipynb`, Chroma DB 등을 활용한 체인 구성 및 RAG(검색 증강 생성) 연습
* **챗봇 및 프롬프트 실험**: `chatbot.py` 및 날짜별 연습용 주피터 노트북(`20260318.ipynb` 등)
* **음성/텍스트 처리 (STT/TTS 등)**: `opic.py`, `opic.ipynb` 및 오디오 파일(`.mp3`, `.wav`)을 활용한 테스트
* **RAG 테스트 문서**: `현진건-운수좋은날+B3356-개벽.pdf` 등 로컬 문서를 활용한 데이터 추출 및 질의응답 연습

##  주로 사용하는 환경
* Python, Jupyter Notebook
* LangChain, ChromaDB
